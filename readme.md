## Usage

```
$ git clone https://github.com/kobatakuJP/statsjstest.git
$ cd statsjstest
$ npm install
```

### stats.js test

```
$ npm run build-d
$ npm run server
```

see localhost:8080 from your browser.

![stats.js](./img/stats.js.png)

### stats-js test

```
$ npm run build-h
$ npm run server
```
